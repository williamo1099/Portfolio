# Portfolio

My personal portfolio project built using **[Next.js](https://nextjs.org)**, bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started

### 1. Clone the Repository

Clone the repository to your local machine:

```bash
git clone https://github.com/williamo1099/Portfolio
cd your-project
```

### 2. Install Dependencies

Install the required frontend dependencies:

```bash
npm install
```

### 3. Run the Development Server

Start the development server to preview your portfolio locally:

```bash
npm run dev
```

### 4. Build for Production

To generate a production-ready build, run:

```bash
npm run build
npm start
```
